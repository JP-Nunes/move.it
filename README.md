## If you want to run it (Pretty Simple):
  * ### Required technologies:
    * [Node.JS](https://nodejs.org/en/)
    * [Yarn](https://yarnpkg.com/getting-started/install)
    
  * Steps to run it:
    * Inside the project's root folder, run "yarn";
    * Run "yarn dev", that's it;
